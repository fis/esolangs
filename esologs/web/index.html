about

This server hosts the logs associated with the esolangs.org community IRC channel(s), wherever they may be. See the Esolang:Community portal page for more information about the channel and the community. The code responsible for collecting and serving the logs can be found at github.com/fis/esolangs.

note (2021-05-29)

We have recently migrated from freenode to the Libera.Chat network as the primary home of the channel. See below for logs from either channel. As of this writing, the channels are still bridged together, so the content is mostly (but not necessarily entirely) redundant.
